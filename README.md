# Pokedex

## Overview
- Static Pokedex UI (list + detail) with no admin UI.
- All Pokemon content and UI copy are in German.
- Data and assets are file-based; the build pipeline turns them into `dist/`.

## Documentation
- dexOS (OS layer, APIs, menus, lifecycle): `public/js/dexOS/README.md`
- Pokedex app (UI + data model details): `public/js/apps/pokedexApp/README.md`

## Key Locations
- Content/data: `data/`
- Public assets: `public/`
- Build output: `dist/`
- Scripts: `scripts/`
